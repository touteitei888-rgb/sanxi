# 散席 · AFTER THE TABLE

> 散席 · 账清人不散

一款极简优雅的聚会 AA 结算工具。聚餐、出游、团建后，一键算清谁该给谁转钱。

## ✨ 功能特性

- 🎉 **创建活动** - 输入活动名称，添加参与人，支持头像和颜色标签
- 💰 **添加账单** - 手动添加消费项目，支持多分类、多垫付人
- 🤝 **多垫付人支持** - 一笔消费可由多人共同垫付，自动拆分
- 🧮 **智能拆账** - 自动计算每个人应付金额，优化转账路径
- 💳 **清晰结算** - 左边付款人，右边收款人，中间金额，一目了然
- 📱 **账单卡片** - 生成精美结算卡片，长按保存图片分享
- 💾 **本地存储** - 数据保存在 localStorage，无需登录，隐私安全
- 🎨 **Ins 风设计** - 玻璃态磨砂质感，大圆角卡片，粉紫渐变

## 🛠️ 技术栈

- **React 19** + **TypeScript**
- **Vite** - 构建工具
- **Tailwind CSS 3** - 样式框架
- **html2canvas** - 账单卡片生成图片
- **vite-plugin-singlefile** - 单文件打包

## 📁 本地开发

```bash
npm install
npm run dev
```

## 📦 构建生产版本

```bash
npm run build
```

构建产物在 `dist/` 目录下，单文件 HTML 可直接打开使用。
