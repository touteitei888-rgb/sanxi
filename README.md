# 散席 · AFTER THE TABLE

> 散席 · 账清人不散
> After the table, settle the bills, keep the friendship.

一款极简优雅的聚会 AA 结算工具。聚餐、出游、团建后，一键算清谁该给谁转钱。

A minimalist and elegant AA bill-splitting tool for group gatherings. Calculate who owes whom with one click after dinners, trips, and team building events.

## ✨ 功能特性 | Features

- 🎉 **创建活动 | Create Events** - 输入活动名称，添加参与人，支持头像和颜色标签
  - Enter event name, add participants with avatars and color tags
- 💰 **添加账单 | Add Bills** - 手动添加消费项目，支持多分类、多垫付人
  - Manually add expense items with multiple categories and multiple payers
- 🤝 **多垫付人支持 | Multiple Payers** - 一笔消费可由多人共同垫付，自动拆分
  - Single expense can be split among multiple payers, auto-calculated
- 🧮 **智能拆账 | Smart Splitting** - 自动计算每个人应付金额，优化转账路径
  - Auto-calculate amounts owed and optimize transfer paths
- 💳 **清晰结算 | Clear Settlement** - 左边付款人，右边收款人，中间金额，一目了然
  - Payer on left, payee on right, amount in between — crystal clear
- 📱 **账单卡片 | Bill Card** - 生成精美结算卡片，长按保存图片分享
  - Generate beautiful settlement cards, long-press to save as image
- 💾 **本地存储 | Local Storage** - 数据保存在 localStorage，无需登录，隐私安全
  - Data stored in localStorage, no login required, privacy-first
- 🎨 **Ins 风设计 | Instagram-style UI** - 玻璃态磨砂质感，大圆角卡片，粉紫渐变
  - Glassmorphism frosted texture, large rounded cards, pink-purple gradient

- **React 19** + **TypeScript**
- **Vite** - 构建工具 | Build tool
- **Tailwind CSS 3** - 样式框架 | Styling framework
- **html2canvas** - 账单卡片生成图片 | Bill card image generation
- **vite-plugin-singlefile** - 单文件打包 | Single-file bundling

## 📁 本地开发 | Local Development

```bash
npm install

```

## 📦 构建生产版本 | Build for Production

```bash
npm run build
```

构建产物在 `dist/` 目录下，单文件 HTML 可直接打开使用。

Build output is in the `dist/` directory. The single HTML file can be opened directly in any browser.
